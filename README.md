# homebrew-sekd

Homebrew tap for [sekd](https://github.com/jefrnc/sekd) — SEC Decoded, a CLI tool for stock due diligence, SEC filing analysis, and dilution risk detection.

## Install

```bash
brew tap jefrnc/sekd
brew install sekd
```

## Usage

```bash
sekd --help
sekd version
```

For full documentation see the [sekd repository](https://github.com/jefrnc/sekd).
